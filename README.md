# Backend Task Manager

## Instalación

npm install

## Ejecutar

npm run dev

## Variables

MONGODB_URI=

JWT_SECRET=

EMAIL_USER=

EMAIL_PASS=

## Endpoints

POST /api/users/register

POST /api/users/login

GET /api/tasks

POST /api/tasks

PUT /api/tasks/:id

DELETE /api/tasks/:id

GET /api/categories

POST /api/categories

PUT /api/categories/:id

DELETE /api/categories/:id
